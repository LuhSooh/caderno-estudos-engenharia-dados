# 📘 Caderno Temático: Introdução à Engenharia de Dados

## Contexto e Objetivos

Este projeto foi desenvolvido como desafio da DIO, utilizando Inteligência Artificial como ferramenta de aprendizagem ativa através do NotebookLM.

O objetivo deste estudo é compreender os fundamentos da Engenharia de Dados, conhecer conceitos importantes da área e organizar meu aprendizado de forma prática.

Durante o projeto serão abordados temas como:

- Banco de Dados;
- SQL;
- ETL;
- Data Lake;
- Data Warehouse;
- Processamento de Dados.

## 📚 Miniguia de Estudos

### Engenharia de Dados

Área responsável por criar estruturas e processos para coletar, armazenar, transformar e disponibilizar dados.

### ETL

Processo de:

- Extract (Extração)
- Transform (Transformação)
- Load (Carregamento)

### Data Lake

Ambiente utilizado para armazenar grandes volumes de dados em diferentes formatos.

### Data Warehouse

Banco de dados estruturado para análises e relatórios.
