# 📚 Miniguia de Estudos: Engenharia de Dados

## O que é Engenharia de Dados?

Engenharia de Dados é a área da tecnologia responsável por criar estruturas para coletar, armazenar, organizar e disponibilizar dados para análise.

## Principais conceitos

### SQL
SQL é uma linguagem utilizada para consultar, organizar e manipular informações armazenadas em bancos de dados.

### ETL
ETL é um processo utilizado para movimentar dados:

- Extract (Extração)
- Transform (Transformação)
- Load (Carregamento)

### Data Lake
É um ambiente utilizado para armazenar grandes volumes de dados em diferentes formatos.

### Data Warehouse
É um banco de dados estruturado usado principalmente para análises e relatórios.

## Glossário

**Pipeline de Dados:** conjunto de etapas que transportam e transformam dados.

**Big Data:** grande quantidade de dados que precisa de tecnologias específicas para processamento.

**Data Engineer:** profissional responsável por construir e manter estruturas de dados.
